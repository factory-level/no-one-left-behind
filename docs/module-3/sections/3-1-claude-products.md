# 3.1 Claude Products Explained

## Web Interface: Claude.ai

### What Makes Claude.ai Unique

- Persistent context through Projects
- Advanced reasoning capabilities
- Nuanced understanding of complex instructions

### Interface Features

- **Projects**: Create specialized workspaces
- **Artifacts**: Interactive document generation
- **Context Management**: Sophisticated memory across conversations

### Product Lineup

1. **Claude.ai (Web)**
   - Daily driver for most tasks
   - Free and pro tiers
   - Best for exploratory work and learning

2. **Claude Desktop**
   - Native application with enhanced features
   - Offline capabilities
   - Deeper system integration

3. **Claude API**
   - Programmatic access
   - For developers and advanced workflows
   - Requires technical setup

4. **Claude Code**
   - Your agent development environment
   - Covered in depth in Module 8

## When to Use Each Product

### Web Interface
- Quick, conversational tasks
- Research and brainstorming
- Learning and exploration

### Desktop
- Consistent work environments
- Large document processing
- Reduced distractions

### API
- Automated workflows
- Integration with other tools
- Batch processing

## Key Considerations

- **Privacy**: Understanding data usage
- **Context Limits**: 200K token window
- **Model Versions**: Sonnet vs Opus differences
