# What is Claude Code

## Section Overview

Claude Code is a powerful CLI tool designed for business professionals to automate workflows, manage files, and create intelligent agent systems without traditional programming skills.

### Key Concepts

- Beyond coding: Automation for everyone
- Native file system integration
- MCP (Model Context Protocol) support
- Sub-agent coordination capabilities

### Learning Objectives

1. Understand Claude Code's core capabilities
2. Recognize non-technical use cases
3. Explore professional automation potential

### Practical Implications

Transform complex business tasks into streamlined, automated processes.
