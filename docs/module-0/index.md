# Module 0: Foundation & Setup

Welcome to your AI automation journey! This foundational module transforms you from an AI curious beginner into someone ready to build intelligent workflows. Think of this as setting up your digital workshop before crafting amazing AI-powered solutions.

## What You'll Accomplish

By the end of this module, you'll have:

**Your AI Command Center Ready**
- Free accounts on all major AI platforms (Claude, ChatGPT, Gemini)
- A properly configured workspace for AI development
- Confidence using basic command-line operations
- Your first successful AI conversation recorded

**Practical Skills Gained**
- Navigate terminal/command prompt without fear
- Organize files in AI-friendly formats
- Create and edit markdown, CSV, JSON, and YAML files
- Set up project folders that AI systems can easily work with

**Mindset Transformation**
- From "AI is intimidating" to "AI is a powerful teammate"
- From manual, one-off tasks to systematic, automated workflows
- From consuming AI content to creating AI solutions

## Learning Objectives

!!! success "By Module Completion"
    **Confidence Level**: You'll feel comfortable opening a terminal and creating your first AI prompt

    **Technical Skills**: Account setup, basic file management, workspace organization

    **Conceptual Understanding**: How AI tools fit into modern work systems

### Module Outcomes Checklist

- [ ] Successfully log into Claude, ChatGPT, and Gemini
- [ ] Execute 5 basic command-line operations
- [ ] Create a properly organized AI project folder
- [ ] Write your first structured prompt and get meaningful results
- [ ] Understand the difference between data formats (CSV vs JSON vs YAML)

## Prerequisites & Expectations

**What You Need**:
- Any computer (Windows 10+, macOS 10.14+, or Linux)
- Reliable internet connection
- 2-4 hours of focused time
- Willingness to try new things

**What You DON'T Need**:
- Programming experience
- Previous AI knowledge
- Expensive software or subscriptions
- Technical background

**Our Promise to You**:
Every technical concept will be explained using everyday analogies. When you feel stuck, remember: every expert was once a beginner, and this course is designed to meet you exactly where you are.

## Module Structure & Time Investment

### 0.1 Course Orientation (30-45 minutes)
**Focus**: Vision and mindset preparation
- Understand the AI landscape and your place in it
- Set realistic expectations for your learning journey
- Complete your first goal-setting exercise

### 0.2 Essential Setup (60-90 minutes)
**Focus**: Hands-on tool installation and account creation
- Create free-tier accounts on major AI platforms
- Install essential tools (VS Code, Git, Terminal setup)
- Master basic command-line operations for Windows and Mac

### 0.3 Understanding Your Workspace (45-60 minutes)
**Focus**: File organization and document formats
- Learn AI-friendly file naming and organization
- Practice with real CSV, JSON, YAML, and Markdown files
- Set up your first AI project workspace

### Module Wrap-up (15-30 minutes)
**Focus**: Knowledge consolidation and readiness assessment
- Complete the Module 0 Readiness Quiz
- Reflect on your setup and learning goals
- Get oriented for Module 1 content

## Success Indicators

You're ready to advance when you can confidently:

1. **Navigate Your Tools**: Open a terminal, create a folder, and edit a file without hesitation
2. **Communicate with AI**: Write clear, structured prompts that get useful responses
3. **Organize Your Work**: Create a project folder structure that supports AI workflows
4. **Troubleshoot Basics**: When something doesn't work, you know where to look for solutions
