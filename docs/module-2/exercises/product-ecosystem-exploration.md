# Product Ecosystem Exploration

**Module 2 Exercise 1 | Duration: 2-3 hours | Assessment Requirement: Set up accounts and complete basic tasks**

## Learning Objectives

By completing this exercise, you will:
- Successfully set up accounts with major AI platforms
- Complete hands-on tasks in Claude.ai, ChatGPT, and Gemini
- Experience the different interfaces and capabilities firsthand
- Develop criteria for evaluating AI tool experiences
- Build confidence using multiple AI platforms

## Overview

This exercise guides you through practical exploration of the three major AI platforms. You'll set up accounts, complete specific tasks, and document your experiences. Think of this as a "test drive" where you get hands-on experience with each tool's strengths and interface style.

**Important:** Take your time with each platform. The goal is understanding, not speed.

## Part 1: Claude.ai Setup & Exploration

### Setup (15 minutes)
1. **Create Account**
   - Go to claude.ai
   - Sign up with email or Google account
   - Verify your email address
   - Complete the welcome flow

2. **Explore the Interface**
   - Notice the clean, conversational layout
   - Find the "Projects" feature (left sidebar)
   - Locate your conversation history
   - Check your usage limits in settings

### Task 1: Business Document Analysis (20 minutes)

**Scenario:** You need to analyze a team meeting transcript for key action items.

**Your Task:**
1. Start a new conversation with Claude
2. Use this prompt:
   ```
   I'm going to share a meeting transcript with you. Please analyze it and provide:
   1. A summary of key decisions made
   2. Action items with responsible parties
   3. Follow-up questions that should be addressed

   Here's the transcript: [Use the sample transcript below]
   ```

**Sample Meeting Transcript:**
```
Sarah: Thanks everyone for joining. We need to finalize the Q4 budget proposal. Marketing is requesting an additional $15K for digital campaigns.

Mike: I can approve that if we see the ROI projections. Can marketing get those to us by Friday?

Sarah: Absolutely. Lisa, can you handle that?

Lisa: Yes, I'll have detailed projections ready. Should I include the competitor analysis we discussed last week?

Mike: That would be helpful. Also, we should revisit the vendor contracts. Some are up for renewal in December.

Sarah: Good point. Let's schedule a separate meeting for that. Mike, can you send a list of expiring contracts?

Mike: Sure, I'll get that out by Wednesday.

Sarah: Great. One more thing - the new hire onboarding process needs updating. Lisa, thoughts?

Lisa: I've been working on that. I should have recommendations by next week's meeting.

Sarah: Perfect. Anything else? No? Thanks everyone.
```

3. Review Claude's response
4. Ask a follow-up question like: "What potential risks do you see in these decisions?"

**What to Notice:**
- How Claude structures its analysis
- The depth of insights provided
- How it handles follow-up questions
- The conversational tone

### Task 2: Project Creation (15 minutes)

1. **Create Your First Project**
   - Click "Projects" in the sidebar
   - Create a new project called "Meeting Analysis Workspace"
   - Add a project description: "For analyzing team meetings and extracting actionable insights"

2. **Configure the Project**
   - Upload a document (create a simple text file with your team structure)
   - Add custom instructions like: "When analyzing meetings, always highlight potential delays and risks"
   - Start a conversation within this project context

3. **Test Project Context**
   - Ask: "Based on the project context, what should I focus on in team meetings?"
   - Notice how Claude incorporates your project setup

**Documentation:** Note how projects feel different from regular chats.

## Part 2: ChatGPT Setup & Exploration

### Setup (15 minutes)
1. **Create Account**
   - Go to chat.openai.com
   - Sign up with email or Google account
   - Verify your email
   - Explore the free tier features

2. **Interface Tour**
   - Notice the chat-focused design
   - Find the conversation history (sidebar)
   - Check settings and preferences
   - Look for "Custom GPTs" option

### Task 1: Brainstorming Session (20 minutes)

**Scenario:** You need to brainstorm ideas for improving team productivity.

**Your Task:**
1. Start a new conversation with this prompt:
   ```
   I need to brainstorm ways to improve productivity for a 5-person remote team. We struggle with:
   - Too many meetings that could be emails
   - Unclear project priorities
   - Information scattered across different tools

   Please suggest 10 creative solutions and explain why each would work.
   ```

2. After receiving the response, ask:
   ```
   Pick the top 3 suggestions and create a detailed implementation plan for each, including timeline and potential obstacles.
   ```

**What to Notice:**
- ChatGPT's creative and conversational style
- How it handles multi-part requests
- The structure and creativity of responses
- Speed of responses

### Task 2: Data Formatting Exercise (15 minutes)

**Your Task:**
1. Give ChatGPT this messy data:
   ```
   Employee data: John Smith, Marketing, joined 2019, salary $65000, manager Sarah Jones. Mary Williams, Sales, joined 2021, salary $58000, manager Mike Chen. David Brown, IT, joined 2020, salary $72000, manager Lisa Park.
   ```

2. Ask it to: "Convert this into a clean CSV format with headers: Name, Department, Start Year, Salary, Manager"

3. Then ask: "Now create a summary report of this data including average salary by department"

**What to Notice:**
- How ChatGPT handles data transformation
- Accuracy in following specific formatting requests
- Ability to perform calculations

## Part 3: Gemini Setup & Exploration

### Setup (15 minutes)
1. **Create Account**
   - Go to gemini.google.com
   - Sign in with your Google account
   - Accept terms and conditions
   - Explore the interface

2. **Interface Exploration**
   - Notice the Google-style clean design
   - Find conversation history
   - Check settings and privacy controls
   - Look for file upload capabilities

### Task 1: Multi-File Analysis (20 minutes)

**Scenario:** You need to analyze multiple documents simultaneously.

**Preparation:**
1. Create 3 simple text files:
   - "Q3_Sales.txt" with: "Q3 sales were $45,000. Top client: ABC Corp ($12,000). Challenges: seasonal dip in August."
   - "Q3_Marketing.txt" with: "Q3 marketing spent $8,000. Digital campaigns: $5,000. Events: $3,000. Lead generation up 15%."
   - "Q3_Operations.txt" with: "Q3 operations ran smoothly. New software implementation delayed to Q4. Team size increased by 2."

**Your Task:**
1. Upload all three files to Gemini
2. Use this prompt:
   ```
   I've uploaded Q3 reports from three departments. Please analyze these documents and provide:
   1. An integrated summary of Q3 performance
   2. Connections between the departments (how one affects another)
   3. Recommendations for Q4 based on all three reports
   ```

**What to Notice:**
- How Gemini handles multiple files simultaneously
- Its ability to find connections across documents
- The comprehensiveness of cross-document analysis

### Task 2: Large Context Test (15 minutes)

**Your Task:**
1. Create a longer document by copying and pasting multiple articles or long text
2. Ask Gemini to:
   ```
   Please read through this entire document and create:
   - A detailed outline of all main topics
   - Key insights that appear multiple times
   - Any contradictions or conflicting information
   ```

**What to Notice:**
- How well Gemini handles large amounts of text
- Its ability to maintain coherence across long content
- Whether it captures details from the entire document

## Troubleshooting Guide

**Problem:** Can't access certain features
- **Solution:** Check if you're on free tier - some features require subscription

**Problem:** Platform seems slow or unresponsive
- **Solution:** Try different browser or clear cache; check internet connection

**Problem:** Responses aren't what I expected
- **Solution:** Refine your prompts - be more specific about what you want

**Problem:** File uploads not working
- **Solution:** Check file size limits and supported formats for each platform
